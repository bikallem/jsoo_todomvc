[Running Demo](https://bikallem.github.io/jsoo_todomvc/)

## Building the project (requires opam v2.1.0 or higher)

```
git clone git@github.com:bikallem/jsoo_todomvc.git
cd jsoo_todomvc
make install
```

## Updating dependencies

`make update`
